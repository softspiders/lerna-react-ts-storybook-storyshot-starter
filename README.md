## Istallation

```sh
yarn
```

or

```sh
npm i
```


## Running

### Test

```sh
npm test
```

### Build

Build lib:

```sh
npm run build:lib
```

Build app:

```sh
npm run build:app
```
### Start

```sh
npm start
```

### Storybook

```sh
npm story
```
