## Istallation

```sh
yarn
```

or

```sh
npm i
```

then

```sh
npm run storybook
```

## Running

### Test

```sh
npm test
```

### Build

Build lib:

```sh
npm run build:lib
```

Build app:

```sh
npm run build:app
```
### Start

```sh
npm start
```

### Storybook

```sh
npm run story
```
